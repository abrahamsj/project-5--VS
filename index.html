Basic Structure of an Array



Introduction


An array is a data structure that serves as a collection of multiple items.
You can iterate over the array or access individual items using an index.




ADVANTAGES


  1. Arrays help in code optimization. We can store a large number of values in a single array by writing a small piece of code rather than declaring each variable separately.

  2. Arrays are easy to use as many algorithms like searching and sorting techniques, finding maximum and minimum values, reversing can be easily implemented using arrays.

  3. The time complexity to access any element of an array is O(1), i.e, it takes a constant amount of time to access an element.



DISADVANTAGES


  1. The size of an array is fixed. Once the memory is allocated to an array, it cannot be increased or decreased. This prevents us from storing extra data in case we want to. These arrays of fixed size are called static arrays.

  2. Allocating less memory than the required to an array leads to loss of data.

  3. The deletion and insertion operations in arrays are very difficult to implement as they store data in contiguous memory locations. To overcome this problem, linked lists are implemented which provide random access of elements.
